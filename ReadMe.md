# [1. 项目简介](#1-项目简介)
   - 本项目旨在以程序化方式生成球形地基和表面植被建筑，其中星球地基由mesh cube归一化得来，表面植被建筑由marchingcube+wfc算法生成。
   - 项目共四个主要流程，分别为原始数据处理，meshgenerating，marchingcube，wfc，以下对各个阶段设计思路进行分析。

  ## 2.2 Mesh Generating
   - **主要流程**：生成顶点，归一化，生成三角索引（某一个面），设置uv，设置法线，生成其他五个面。
   - **面的绘制顺序**：为顺时针，每个四边面由两个三角面组合而成。
     [图片]
   - **极点展开布局**：
     - 解决边缘顶点收缩问题。
     - 不使用球面归一化映射，而使用alternative映射。
     - 参考：https://catlikecoding.com/unity/tutorials/procedural-meshes/cube-sphere/
   - **解决法线接缝问题**：
     1. 共享顶点/焊接顶点。
     2. 修改归一化算法。
     3. 以顶点到原点连成的向量作为法线向量。
